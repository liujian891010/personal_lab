# 存储改造前置准备清单

## 1. 隔离模型确认

- [ ] 确认工作区隔离主键：`userId` / `tenantId` / `orgId`
- [ ] 确认是否采用“一人一空间”
- [ ] 确认是否允许“一个人访问多个空间”
- [ ] 确认管理员是否允许跨空间查看数据
- [ ] 明确后续所有数据表统一使用的隔离字段名称

## 2. 登录返回字段确认

- [ ] 确认 APPKEY 外部登录接口稳定返回 `userId`
- [ ] 确认是否还会返回 `tenantId` / `orgId`
- [ ] 确认用户展示名字段固定取 `userName`
- [ ] 确认缺字段时的兜底规则
- [ ] 形成字段映射表：`外部字段 -> 系统字段`

## 3. 数据库选型与接入

- [ ] 确认生产数据库：`PostgreSQL` 或 `MySQL`
- [ ] 提供数据库连接信息
- [ ] 提供数据库名和账号权限范围
- [ ] 确认是否允许自动执行 schema migration
- [ ] 确认数据库备份策略
- [ ] 确认数据库恢复演练方式

## 4. 对象存储选型与接入

- [ ] 确认对象存储：`OSS` / `S3` / `COS` / `MinIO`
- [ ] 提供 `bucket`
- [ ] 提供 `endpoint`
- [ ] 提供 `region`
- [ ] 提供 `access_key`
- [ ] 提供 `secret_key`
- [ ] 确认对象 key 规范，例如 `workspaces/{workspace_id}/...`
- [ ] 确认是否启用生命周期清理策略

## 5. 数据分层规则确认

- [ ] 明确哪些数据进入数据库
- [ ] 明确哪些数据只进入对象存储
- [ ] 明确哪些数据允许只做本地临时缓存
- [ ] 明确删除记录时是否同步删除对象存储文件
- [ ] 明确是否需要软删除和回收站

## 6. 上传与容量规则

- [ ] 确认单文件大小上限
- [ ] 确认单用户总容量上限
- [ ] 确认单用户文件数量上限
- [ ] 确认支持的文件类型清单
- [ ] 确认 OCR / 文本提取页数限制
- [ ] 确认是否需要大文件分片上传

## 7. 生命周期与清理规则

- [ ] 确认临时文件保留时长
- [ ] 确认失败任务文件保留时长
- [ ] 确认删除文件后是否进入回收站
- [ ] 确认回收站保留时长
- [ ] 确认是否需要定时清理任务
- [ ] 确认清理动作是否需要审计日志

## 8. 安全与权限规则

- [ ] 确认服务端是否必须二次校验 workspace 权限
- [ ] 确认所有查询是否默认强制带 workspace 过滤条件
- [ ] 确认是否需要上传文件病毒扫描
- [ ] 确认是否需要敏感文件加密存储
- [ ] 确认是否需要操作审计日志
- [ ] 确认是否需要访问频率限制

## 9. 历史数据迁移方案

- [ ] 盘点现有本地数据范围
- [ ] 确认是否需要回填历史文件到对象存储
- [ ] 确认是否需要迁移旧 SQLite 数据
- [ ] 确认迁移窗口
- [ ] 确认迁移失败时的回滚方案
- [ ] 确认迁移完成后的校验标准

## 10. 测试与验收准备

- [ ] 提供至少 2 个真实用户 APPKEY
- [ ] 如果有租户维度，再提供跨租户 APPKEY
- [ ] 准备 `md` / `pdf` / `docx` 测试文件
- [ ] 准备大文件样本
- [ ] 准备中文文件名样本
- [ ] 明确验收标准：用户 A 不能看到用户 B 数据

## 11. 运维环境准备

- [ ] 确认部署环境：开发 / 测试 / 生产
- [ ] 确认环境变量管理方式
- [ ] 确认日志采集方式
- [ ] 确认监控告警方式
- [ ] 确认磁盘、数据库、对象存储容量预估
- [ ] 确认域名 / CORS / 反向代理规则

## 12. 需要优先给出的最小输入

- [ ] 最终隔离主键选型
- [ ] 生产数据库类型
- [ ] 对象存储类型
- [ ] 对应连接配置
- [ ] 配额和删除规则
- [ ] 2 个以上测试 APPKEY

## 13. 交付建议顺序

1. 先确认隔离主键：`userId` 还是 `tenantId/orgId`
2. 再确认生产数据库
3. 再确认对象存储
4. 再提供联调配置和测试账号
5. 最后确认配额、删除、清理规则
