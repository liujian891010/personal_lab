# Phase 7: 运维、安全与上线 Runbook

## 目标

确保改造后的存储方案具备可上线、可审计、可回滚、可观测能力。

## 改造步骤

- 凭证与权限
  - 对象存储使用最小权限 AK/SK 或角色
  - 区分读写桶、公开桶、私有桶
- 安全策略
  - 默认私有读
  - 下载走签名 URL 或服务端代理
  - 敏感文件访问写审计日志
- 可观测性
  - 记录对象上传失败率
  - 记录对象读取耗时
  - 记录迁移成功率
  - 记录磁盘缓存占用
- Runbook
  - 环境变量清单
  - 初始化桶与目录前缀
  - 灰度发布步骤
  - 回滚步骤
  - 故障排查步骤
- 验收标准
  - 新上传不长期占本地盘
  - 老数据迁移可校验
  - 用户隔离保持不变
  - 异常情况下可回滚

## 最终交付

- 技术设计文档
- 迁移脚本
- 运维手册
- 上线检查表
